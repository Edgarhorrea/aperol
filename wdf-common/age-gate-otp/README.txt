Add the following snippet inside the “head tag” of your website 
adding the site name in data-domain-path variable (e.g. examplewebsite.com) and in “src” the path to the “age-gate-otp.js”

Eg.
<script id="ageGateOTP" data-domain-path="examplewebsite.com" src="/wdf-common/age-gate-otp/age-gate-otp.js"></script>
